# ChickenDetecter
KDDI 学生向けオンラインハッカソン KDDI DIGITAL GATE HACKS(KDG HACKS) 2022 SUMMER でチーム開発した Web アプリ。

怪談話を聞いているミーティング動画を解析して、ビビりが誰かを決められるというサービス。
